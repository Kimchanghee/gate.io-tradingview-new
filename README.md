# Gate.io TradingView Trading Bot

자동화된 Gate.io 거래 봇 - TradingView 웹훅 신호를 받아 자동으로 거래를 실행합니다.

## 🚀 주요 기능

- TradingView 알림을 통한 자동 거래
- Gate.io 현물 거래 지원
- 실시간 웹소켓 모니터링
- 리스크 관리 시스템
- 관리자 대시보드
- Telegram/Discord 알림
- 거래 로그 및 통계

## 📋 요구사항

- Node.js 14.0 이상
- Gate.io API 키 (거래 권한 필요)
- TradingView 계정 (Pro 이상 권장)
